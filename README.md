# 🪨📄✂️ Stone Paper Scissor Game (Python)

A simple command-line **Stone Paper Scissor Game** built using Python and NumPy. The user plays against the computer for a specified number of rounds, and the final winner is determined based on the scores.

## 🚀 Features

* Play multiple rounds against the computer.
* Random computer choices using NumPy.
* Score tracking for both user and computer.
* Displays the overall winner at the end.
* Handles invalid user inputs.

## 🛠️ Technologies Used

* Python 3
* NumPy

## ▶️ How to Run

Run the Python file:

```bash
python stone_paper_scissor.py
```

## 🎮 Game Rules

* Stone beats Scissor
* Scissor beats Paper
* Paper beats Stone
* If both choices are the same, the round is a Draw
